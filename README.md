# MooCareX Muzzle Enrollment System

AI-powered cattle biometric enrollment platform.

## Features

- Muzzle Detection (YOLO)
- Quality Classification
- Mobile Camera Capture
- Metadata Storage (JSON)
- User Feedback Collection

## Tech Stack

- React + Vite
- FastAPI
- YOLO
- OpenCV

## Run Frontend

npm install
npm run dev

## Run Backend

uvicorn main:app --reload
